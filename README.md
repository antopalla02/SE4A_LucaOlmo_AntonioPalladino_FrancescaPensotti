## Project Description
FreelanceMatch is a web-based platform that automates the matching between freelancers and clients. A client publishes a project with required skills, budget and deadline; the system automatically computes a compatibility score for each registered freelancer and returns a ranked list of the most suitable candidates. Freelancers receive targeted notifications about projects that match their profile. The platform also manages the full project lifecycle: from publication, to proposal management, up to mutual reviews at the end of the collaboration.
